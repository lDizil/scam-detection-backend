# Дообучение модели

Эта папка содержит скрипты и данные для дообучения модели на ваших собственных данных.

## Структура

```
training/
├── data/              # Данные для обучения
│   ├── train.csv     # Обучающая выборка
│   └── test.csv      # Тестовая выборка
├── models/           # Сохраненные дообученные модели
└── train.py          # Скрипт для обучения
```

## Формат данных

Файлы CSV должны иметь следующую структуру:

```csv
text,label
"Срочно! Ваш аккаунт заблокирован",1
"Привет, как дела?",0
```

Где:

- `text` - текст сообщения
- `label` - метка класса (0 = legitimate, 1 = phishing)

## Как дообучить модель

1. Подготовьте данные в формате CSV и положите их в `training/data/`

2. Запустите скрипт обучения:

```bash
python training/train.py
```

3. Дообученная модель сохранится в `training/models/`

4. Обновите `.env` файл, указав путь к новой модели:

```
CUSTOM_MODEL_PATH=./training/models/my_finetuned_model
```

5. Перезапустите ML сервис

## Рекомендации по данным

- Минимум 1000 примеров для обучения
- Сбалансированное соотношение классов (50/50)
- Разнообразные примеры мошенничества:
  - Фишинг банков
  - Поддельные призы
  - Запросы данных карт
  - Срочные переводы
  - Технические поддержки

## Метрики качества

После обучения скрипт выведет:

- Accuracy (точность)
- Precision (точность positive класса)
- Recall (полнота)
- F1-Score (гармоническое среднее)

Целевые метрики для production:

- Precision > 0.90 (мало ложных срабатываний)
- Recall > 0.85 (детектим большинство мошенничеств)
